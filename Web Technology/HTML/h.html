HTML-Hypertext MarkUp Language

HTML is a Standard markup language for Web Pages.

With Html we can create own Website

It is a code used to structure a web page and its Content

For Example, Content could be structured within a set of paragraphs, a list of bulleted points, or using images and data tables
There are range of HTML tags, they help you to design web page. There are four required tags in HTML. These are html, title, head and body
